# Data Jobs Dashboard w/ Power BI
![Dashboard Page 1](/Images/Proj1img1.png)
## Introduction
This dashboard is built for **job seekers, career switchers, and professionals** exploring new opportunities in the data field. It addresses a common problem: data job market information is spread across multiple sources and is hard to understand at a glance.
Using a *real-world dataset of 2024 data science job postings* that includes job titles, salaries, and locations, this project brings everything together in one interactive dashboard. It allows users to quickly explore market trends, compare roles, and understand compensation patterns in the data job market.

## Dashboard Overview
The report consists of two pages, combining a quick summary view with an in-depth analysis.

### Page 1: High-level Market View

![Dashboard Page 1](/Images/Proj1img1.png)

This page provides a high-level snapshot of the data job market, presenting key KPIs such as total job count, median salaries, and top job roles. It is designed to help users quickly understand overall market trends and compensation patterns at a glance.
### Page 2: Job Title Drill Through

![Dashboard Page 1](/Images/Proj1img2.png)

This page provides a detailed drill-through view from the main dashboard, focusing on a single job title. It breaks down salary ranges, work-from-home availability, benefits indicators, top hiring platforms, and the global distribution of jobs.

## Conclusion
This dashboard demonstrates how Power BI can transform raw job posting data into a clear and actionable career analysis tool. Through interactive filtering, slicing, and drill-through features, users can explore job trends, salary patterns, and role-specific insights, enabling more confident and informed career decisions.
